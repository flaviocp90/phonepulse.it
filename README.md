# PhonePulse

Webapp tech italiana — recensioni e guide smartphone per scegliere bene.

**Stack:** React + Vite + Tailwind CSS + Supabase + Vercel

**Sito:** [phonepulse.it](https://phonepulse.it)

## Setup locale

```bash
npm install
cp .env.local.example .env.local
# Inserisci le credenziali Supabase in .env.local
npm run dev
```
